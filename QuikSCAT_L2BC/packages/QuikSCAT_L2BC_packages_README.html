1- Intoduction


This README contains the instructions to compile all the 2nd party packages required to compile the programs and libraries for the QuikSCAT_L2BC project. All packages are installed assuming that the base directory is the QuikSCAT_L2BC base directory. In the documentation below the root directory is assumed to be given by the environment variable L2BCDIR. For instance, under tcsh you can set


setenv L2BCDIR /Users/er/Scatterometry/QuikSCAT/QuikSCAT_L2BC/


Change accordingly for your location and shell.


2-Blitz


The blitz library is used as the default library for array handling. Current library version is 0.9, which can be downloaded from http://sourceforge.net/projects/blitz/files/. The following instructions are given by compilation platform.

To compile blitz from the ./packages directory:


tar xvzf blitz-0.9.tar.gz

cd blitz-0.9

configure --prefix=$L2BCDIR

make install


This should install the library correctly. After finishing, you can remove the blitz-0.9 directory.


3- Scons


The build system for this project is scons http://www.scons.org. The current version used is 2.0.0.final.0. It is assumed that a working python distribution (version > = 2.4) is available. 


To install a local version of scons:


tar xvzf scons-2.0.0.final.0.tar.gz

cd scons-2.0.0.final.0

python setup.py install --prefix=$L2BCDIR


This should install scons under $L2BCDIR/bin.


4- NetCDF


To make files platform independent, and to allow for future expansions of data fields, the output results are stored in netCDF format ( http://www.unidata.ucar.edu/software/netcdf/ ). 


However, in order to reduce dependencies on the HDF5 and other libraries, netcdf-4.1.1.1 is installed without hdf5 support, dap support, or fortran support. This may change in the future.


To install version 4.1.1 of netcdf:


tar xvzf netcdf-4-1.1.tar.gz

cd netcdf-4.1.1

./configure --prefix=$L2BCDIR --disable-netcdf-4 --disable-dap --disable-fortran --disable-f90

make check install


5- HDF4


HDF4 is used for backward compatibility with the QuikSCAT L2B data. HDF4 requires both lib and the jpeg libraries, which are also included for a self-contained package.


To install zlib:


tar xvzf zlib-1.2.5.tar.gz

cd zlib-1.2.5

./configure --prefix=$L2BCDIR

make install  


To install jpeg:


tar xvzf jpegsrc.v6b.tar.gz

cd jpeg-6b

./configure --prefix=$L2BCDIR

make install

make install-lib 


To install HDF4 (after installing zlib and jpeg. NetCDF support is disabled due to incompatibilities between HDF4.2.5 and NetCDF4):


tar xvzf hdf-4.2.5.tar.gz

cd hdf-4.2.5

./configure --prefix=$L2BCDIR --with-zlib=$L2BCDIR --with-jpeg=$L2BCDIR --enable-fortran=no --enable-netcdf=no

make install


6- GSL


The GNU Scientific Library GSL is used for fitting derivative surfaces to calculate the vector fields curl and divergence.


To install GSL:


tar xvzf gsl-1.14.tar.gz

cd gsl-1.14

./configure --prefix=$L2BCDIR

make

make install
